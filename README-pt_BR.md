# fix-meteor-bcrypt

###### Ao ligar o meteor você recebe esta mensagem?

----

> (STDERR) Note: you are using a pure-JavaScript implementation of bcrypt.

> (STDERR) While this implementation will work correctly, it is known to be

> (STDERR) approximately three times slower than the native implementation.

> (STDERR) In order to use the native implementation instead, run

> (STDERR)

> (STDERR)   meteor npm install --save bcrypt

> (STDERR)

> (STDERR) in the root directory of your application.

---
Então está é a solução para o seu problema!

Tutorial

1. Faça um git clone deste repositório.

2. Abra o prompt de comando como administrador na pasta fix-meteor-bcrypt

3. Basta executar o comando abaixo:

> fix-meteor-bcrypt C:\Caminho\Absoluto\Para\Seu\Projeto

> **Nota:** É muito importante que o prompt de comando esteja como administrador.

Agora é só esperar o comando ser concluído, e após terminado, tudo estará corrigido!
